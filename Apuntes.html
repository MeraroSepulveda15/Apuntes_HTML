Etiquetas para incrustar contenido

img: Inserta una imagen

iframe: Nos sirve para incrustar un documento HTML dentro del documento HTML principal

video: Permite insertar videos dentro del documento, ofreciendo además un interfaz de control del video con los elementos típicos: reproducción, parada, control de volumen, etc…

audio: Permite insertar audios dentro del documento, ofreciendo además un interfaz de control del audio con los elementos típicos: reproducción, parada, control de volumen, etc…

Etiquetas tablas

table: Una tabla muestra un conjunto de elementos relacionados en forma de matriz. Las tablas se construyen de izquierda a derecha (por columnas) y de arriba a abajo (filas).

thead: Indica el bloque de filas o la fila con la información de la cabecera

tbody: Indica el bloque de filas o la fila con la información del cuerpo

tfoot: Indica el bloque de filas o la fila con la infomración de resumen

tr: Se utiliza para indicar una fila de celdas en una tabla

td: Se utiliza para indicar una celda

th: Se utiliza para definir el encabezado

Etiquetas de fomrulario

form: Representa una sección de un documento que contiene controles interactivos que permiten a un usuario enviar información a un servidor web

input: Se utiliza para crear controles interactivos para formularios basados en la web con el fin de recibir datos del usuario

button: Representa un elemento cliqueable de tipo botón que puede ser utilizado en formularios o en cualquier parte de la página que necesite un botón estándar

textarea: Representa un control para la edición mutilínea de texto sin formato

select: Representa un control que muestra un menú de opciones

option: Las opciones contenidas en el menú
